# 브레인스토밍 요약: 회의 로그라이크

날짜: 2026-01-27

## 큰 방향
- "휴지통 던지기" 대신 완전히 새로운 로그라이크
- 턴제 전술 기반 덱빌딩 로그라이크
- 배경: 사무실/회의실 풍자
- 주인공: 회의에 치인 직장인 도트 캐릭터

## 게임 진행 구조
- 하나의 회의실 안에서 스파이어식 노드 선택으로 진행
- 노드 종류: 전투/휴식/상점
- 런 길이 선택 가능, 기본은 45-60분(8-10층)

## 전투/자원
- 에너지 3, 손패 5(스파이어 기본형)
- 핵심 자원: 집중력
  - 턴마다 자동 감소 + 피격 시 추가 감소
  - 회복은 휴식 노드에서만
- 보스 승리 조건: 저항 게이지 감소 + 합의 게이지 상승 동시 달성

## 전술/좌석(노드)
- 회의실 내 자리/노드 효과 중심
- 노드 배치는 고정형

## 카드/덱
- 직업별 덱 차별화
- 전투 후 카드 3장 중 1장 선택
- 카드 태그 혼합(직장 화법/완화/데이터 등)

## 직업/메타 진행
- 기본 클래스: PM, 분석가, 엔지니어, 디자이너
- 메타 진행으로 언락
- 언락 방식: 성과 점수 누적 + 특정 업적

## 적/이벤트
- 적: 인물 유형 + 회의실 이벤트 혼합
- 이벤트 중심 축: 시간 압박
  - 전투당 턴 제한
  - 맵 공용 타이머
  - 실패 시 집중력 대량 감소

## 아트/사운드
- 8비트 레트로 도트 스타일
- 칩튠 사운드

## 라이브포인트 연동
- 회의 점수 산정: 발화량 + 요지 품질 + 침묵 회피(가중치)
- 보상: 카드팩/메타 재화

## 접근 방식
- 접근 A(코어 루프 MVP 우선) 선택
