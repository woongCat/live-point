# 회의 로그라이크(턴제 전술 덱빌딩) 디자인

날짜: 2026-01-27

## 개요
라이브포인트 앱 내 독립 모드로 동작하는 턴제 전술 덱빌딩 로그라이크. 1회 플레이(런)는 하나의 "회의"를 의미하며, 슬레이더 스파이어식 노드 맵을 따라 전투/휴식/상점을 진행한다. 주인공은 "회의에 치인 직장인" 도트 캐릭터.

## 목표
- 카드 덱빌딩 + 좌석(노드) 전술 + 집중력 자원 관리의 긴장감을 동시에 제공
- 직장인 풍자의 테마를 전투/이벤트/보스 패턴으로 녹임
- 회의 기능과 연동되는 보상을 통해 앱 사용 동기를 강화

## 코어 루프
1) 런 길이 선택(선택지 제공, 기본은 45-60분/8-10층)
2) 회의 맵 생성(전투/휴식/상점 노드)
3) 전투 진입 시 고정 회의실 노드 배치 로드
4) 카드 플레이 + 자리 이동으로 저항 감소/합의 증가
5) 전투 승리 후 카드 3장 중 1장 선택
6) 휴식 노드에서만 집중력 회복
7) 보스 설득전 승리 후 런 종료 보상

## 런 구조(회의 맵)
- 노드 타입: 전투/휴식/상점
- 맵은 하나의 회의를 의미하되, 스파이어 스타일 경로 선택 구조
- 공용 타이머 적용: 노드 선택 지연 시 피로 스택 부여

## 전투 규칙
- 기본 턴 구조: 에너지 3, 손패 5 (슬레이더 스파이어 기본형)
- 핵심 자원: 집중력
  - 턴마다 자동 감소
  - 피격 시 추가 감소
  - 휴식 노드에서만 회복
- 승리 조건(보스): 저항 게이지 감소 + 합의 게이지 상승을 모두 달성
- 턴 제한: 전투당 제한 턴 존재
  - 실패 시 집중력 대량 감소(즉시 패배는 아님)

## 좌석(노드) 전술
- 고정 좌석 배치 사용
- 노드 효과 예시:
  - 화이트보드: 데이터 카드 효과 +1
  - 정면 좌석: 압박 카드 추가 피해
  - 구석: 집중력 소모 -1, 발언력 -1
- 이동은 턴당 1회, 에너지 1 소모

## 카드/덱 시스템
- 전투 후 카드 3장 중 1장 선택(강화/확장)
- 카드 태그: 설득/압박/데이터/완화
- 클래스별 덱 차별화
  - PM: 조율/정리 중심
  - 분석가: 데이터 시너지
  - 엔지니어: 문제 해결/일발 역전
  - 디자이너: 설득/완화 중심
- 클래스는 메타 진행으로 언락

## 적(상대) + 환경 이벤트
- 인물 유형 + 회의실 이벤트 혼합
- 인물 보스 예시: 마이크로매니저/카리스마/숫자집착/감성형
- 환경 이벤트 중심 규칙: 시간 압박
  - 전투 턴 제한 강화
  - 맵 공용 타이머로 피로 스택 증가

## 메타 진행(언락/보상)
- 재화 3종:
  - 인사평가 점수(런 성과)
  - 회의 생존지수(집중력 관리)
  - 사내 포인트(상점 통화)
- 클래스 언락: 성과 점수 누적 + 특정 업적 달성

## 라이브포인트 연동 보상
- 회의 점수 계산: 발화량/요지 품질/침묵 회피 가중합
- 회의 점수 보상: 카드팩/영구 패시브 재화 지급
- 게임 성과는 회의 기능의 코스메틱에만 영향

## 아트/사운드
- 도트 그래픽: 8비트 레트로 톤
- 사운드: 칩튠 기반

## MVP 범위(접근 A)
- 고정 회의실 노드 1종
- 노드 3종(전투/휴식/상점)
- 보스 1명
- 클래스 2개부터 시작
- 카드 25-35장

## 테스트 포인트(TDD)
- 전투 턴/에너지/손패 규칙
- 집중력 감소 규칙
- 좌석 노드 효과 적용
- 턴 제한 패널티
- 보스 승리 조건(저항/합의 동시 달성)
- 카드 보상 선택 로직
- 맵 생성/런 길이 선택 일관성

## 에러/폴백
- 회의 점수 계산 실패 시 기본 가중치로 폴백
- 보상 지급 실패 시 최소 보상 제공
- 맵 타이머는 로컬 기준 처리
